# TRS-80_TC-8 Documents Folder

Documents and publications related to JPC Products TC-8 Tape System for the TRS-80 Model I

## Thanks, and Credits

I would first like to thank Pascal Holdry, who compiled several of these documents,
and I hope that people can visit his site where there is history of not just the
TC-8, but many other TRS-80 items as well (In France !):

https://prof-80.fr/index.php/tc-8-jpc-products-company

## Datasheets Folder:

One of the documents below contains a schematic diagram and parts list for the TC-8 system;
I have compiled datasheets for each of the main parts used in the system, and placed them
all in this folder.
